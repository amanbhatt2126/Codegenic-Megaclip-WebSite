About

MegaClip is a result of joint collaboration between Aman Bhatt, Suparv Lakhanpal and Vibhanshu Kaushal. The work was complied under the team name "Codegenic". We intend to present quality games, ranging from endless runner to a mario clone. The following technologies were put into use to develop this mini-project of ours.
1) Visual Studio Code
2) Unity game engine
3) HTML
4) CSS
